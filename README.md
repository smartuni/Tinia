# Tinia

### Inhalt des Projekts
Wenn es draußen stürmt oder uns der Wind langsam durchs Haar streicht, dann hat nur einer die Finger im Spiel: Der Gott Tinia. Was er scheinbar zufällig geschehen lässt, mal launig, mal sanft, wollen wir in Zahlen fassen. Wir fragen uns: Sind Götter messbar? Das RIOT-Projekt Tinia ist die Antwort auf eine geschlossene wissenschaftlich Lücke, die wir wieder öffnen wollen - ähnlich einer ägyptischen Pyramide, in der eine neue Kammer entdeckt wurde. Ein Annäherungsversuch an eine göttliche Macht, die schwer zu greifen ist. Gemeinsam mit dem mächtigen Betriebssystem RIOT entsteht ein Experiment am Rande des Wahnsinns - im transzendenten Raum zwischen Himmel und Erde.
